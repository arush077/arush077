## **📌 Featured Projects**

### 🔀 [MergePilot](https://merge-pilot.vercel.app/)
Autonomous GitHub PR creator powered by a **5-agent pipeline** (IssueAnalyzer, CodebaseResearcher, FixDrafter, TestWriter, PRCreator) that resolves GitHub issues end-to-end — reducing resolution time from hours to seconds. Built with Python, FastAPI, React, and the Groq API, with 3-tier fault tolerance for reliable autonomous execution.

### 📄 [GetHired](https://get-hired-weld.vercel.app/)
Full-stack AI-powered resume builder with multiple templates, drag-and-drop reordering, and real-time PDF export. Node.js + MongoDB backend with JWT auth, integrated with Groq LLM for one-click bullet generation and resume quality analysis — cutting drafting time by 50%+.
